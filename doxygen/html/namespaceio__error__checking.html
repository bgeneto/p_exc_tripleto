integer function, public io_error_checking::free_unit ( integer, optional  unt_min,
integer, optional  unt_max 
)

Finds the first available (not used) unit number from min value to a max value (if given).

Can be used with IO file operations where an unit number is required. This way one can avoid the usage of an explicit unit number while performing operations like open, read/write and close.

Parameters
[in]unt_min(Optional) Starts searching for a free unit number beginning with this value (defaults to 20).
[in]unt_max(Optional) Stops searching for free unit numbers at this value (defaults to 90).
Warning
Program execution is aborted if a (free) unit number cannot be found.
Returns
An integer: the first (free) unit number found.

Checks if an error occurred while reading, writing, opening or closing external files.

This routine tries to print (in the standard output unit) some useful information relative to the triggered error, like the error code, the error message, and the corresponding file name.
